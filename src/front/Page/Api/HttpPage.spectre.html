Формат ответа
Элемент возвращаемого списка имеет формат: 
        
{`{
    "name": "AMD Ryzen Threadripper 3970X",
    "image": "https://hardprice.ru/images/p/1/1ft1bet.jpg",
    "prices": [
        {
            "type": "average",
            "value": 185485,
            "currency": "RUB",
            "precision": 0
        }
    ],
    "vb_ratio": "34.62",
    "benchmarks": [
        {
            "name": "PassMark",
            "value": 64223
        }
    ]
}`}
prices[0].type — строка, название агрегирующей функции или код магазина, пример: regard, citilink, oldi.
prices[0].value — целое число, нормализованное значение стоимости (до применения точности).
prices[0].currency — строка, код валюты по ISO 4217.
prices[0].precision — целое число, определяет положение запятой в значении цены, устанавливая точность денежной суммы. Например, данную запись: 
        
{`{
    "type": "regard",
    "value": 7580021,
    "currency": "RUB",
    "precision": 2
}`}
следует читать как "75800,21 RUB".

Ответ на некоторые запросы может содержать дополнительное поле с датой, информирующей, на какой момент времени актуальны предложенные цены и бенчмарки.
