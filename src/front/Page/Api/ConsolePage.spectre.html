Для использования возможностей командной строки необходимо загрузить и собрать свой экземпляр приложения по данной инструкции.

Расчёт рейтинга комплектующих
Выводит таблицу с соотношением цена/производительность для доступных комплектующих: 
$ docker-compose run --rm --no-deps app bin/console ratio:calculate
Опции:

--source=live — искать свежую информацию через интернет (по умолчанию).
--source=database — переключить на локальное хранилище (будут использованы наиболее актуальные данные, собранные существующим процессом в ходе работы).

Получение бенчмарков
Выводит отдельную таблицу с бенчмарками: 
$ docker-compose run --rm --no-deps app bin/console benchmark:list

Просмотр цен
Выводит таблицу с ценами на комплектующие (fallback): 
$ docker-compose run --rm --no-deps app bin/console price:list
