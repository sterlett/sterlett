www.cpu-junkie.ru — микросервис, анализирующий соотношение цена/производительность для компьютерных комплектующих, подгружая цены из магазинов г. Москвы в реальном времени. С его помощью можно получить информацию, какой чип сейчас выгодно купить и где.

Также его можно использовать для создания архива с ценами на железо и собственной аналитики (см. GitHub).

 

Основное отличие — в формуле используются действующие цены в рублях из российских магазинов, а не абстрактные N долларов. Таким образом, формируется рейтинг комплектующих с учётом плавающей региональной наценки. Сервис старается учитывать только товары в наличии, поэтому шанс увидеть в рейтинге мифическую или устаревшую модель минимален (но не исключается).

Отдельные магазины могут увеличить цену на популярный чип за счёт повышенного спроса среди майнеров, скачка курса валют или банального хайпа. Сервис помогает отслеживать такие ситуации в реальном времени и показывает достойные альтернативы для покупки (а также конкретные магазины).
Если при этом сам список подгружается, скорее всего публичных офферов по нему нет в большинстве известных магазинов Москвы (ИП и доски объявлений не учитываются). Другой вариант — Пассмарк еще не собрал достаточную статистику по этому чипу.

Возможен и баг в парсерах — разрешается создать issue на GitHub.
См. API секцию. На данный момент можно получить список средних цен на процессоры (с посчитанным рейтингом) через эндпоинт:

https://cpu-junkie.ru/api/cpu/list.json

Или поднять свой инстанс. Данные актуализируются каждые 12 часов. Есть консольный интерфейс.
