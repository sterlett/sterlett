www.cpu-junkie.ru — микросервис, анализирующий соотношение цена/производительность для компьютерных комплектующих, подгружая цены из магазинов г. Москвы в реальном времени. С его помощью можно получить информацию, какой чип сейчас выгоднее всего купить и где.

Также его можно использовать для создания архива с ценами на железо и собственной аналитики (см. GitHub).

 

Основное отличие — в формуле используются действующие цены в рублях из российских магазинов, а не абстрактные N долларов. Таким образом, формируется рейтинг комплектующих с учётом плавающей региональной наценки. Сервис старается учитывать только товары в наличии, поэтому шанс увидеть в рейтинге мифическую или устаревшую модель минимален (но не исключается).

Отдельные магазины могут увеличить цену на популярный чип за счёт повышенного спроса среди майнеров, скачка курса валют или банального хайпа. Сервис помогает отслеживать такие ситуации в реальном времени и показывает достойные альтернативы для покупки (а также конкретные магазины).
Это категории производительности комплектующих по CPU Junkie, основанные на PassMark диапазонах.

Процессоры:
A+: 40000+
A: 30000 — 40000
B: 25000 — 30000
C: 20000 — 25000
D: 15000 — 20000
E: 10000 — 15000
F: 0 — 10000

Чем выше категория, тем дороже стоит единица вычислительной мощности. Для более продвинутых камней покупательский Price/Performance ratio может оказаться значительно ниже.

F, E и D — повседневные задачи, C и выше - профессиональное использование.
Если при этом сам список подгружается, скорее всего публичных офферов по нему нет в большинстве известных магазинов Москвы (ИП и доски объявлений не учитываются). Другой вариант — Пассмарк еще не собрал достаточную статистику по этому чипу.

Возможен и баг в парсерах — разрешается создать issue на GitHub.
Стоимость комплектующих отслеживается через веб-ресурс www.hardprice.ru. В перспективе могут быть добавлены дополнительные источники, включая кастомизированные парсеры под конкретные сайты магазинов.

Для расчётов используется стоимость OEM версии, если её нет в продаже — ритейлерская (BOX).

Провайдером бенчмарков выступает австралийская компания PassMark Software (www.passmark.com).

В таблицах (там, где применимо) представлен наиболее общий показатель multiple cores, сравнивающий железо в рамках профессиональных задач (3D-моделирование, рендеринг, стриминг). Существует также single thread дивизион, где Intel чипы начинают отыгрывать свои прайс теги в глазах геймеров, но на данный момент он не учитывается.

Все интеграции являются неофициальными, сайт не аффилирован с какой-либо компанией, предоставляющей данные для аналитики в открытом доступе.
См. API секцию. На текущий момент можно получить список средних цен на процессоры (с посчитанным рейтингом) через эндпоинт:

https://cpu-junkie.ru/api/cpu/ratio.json

Или поднять свой инстанс. Данные актуализируются каждые 12 часов. Есть консольный интерфейс.
